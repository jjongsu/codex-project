# 트렌드 맞춤 3종 추천안

## 목적

- 기존 레트로 감성 방향은 유지하되, 2026년 기준 브라우저/캐주얼 게임에서 더 잘 먹히는 조작 방식과 반복 구조를 반영한다.
- `즉시 플레이`, `짧은 세션`, `모바일 대응`, `점수 경쟁`, `혼자 개발 가능한 범위`를 동시에 만족하는 3종을 확정한다.

## 선정 기준

- 첫 입력까지 5초 이내에 진입 가능한가
- 모바일에서 `탭`, `드래그`, `스와이프` 중심으로 조작할 수 있는가
- 3~7분 세션으로 반복 플레이가 자연스러운가
- 점수와 랭킹 경쟁이 잘 붙는가
- `Phaser + React + Zustand + Supabase` 구조에서 무리 없이 구현 가능한가
- 공략 글, 팁 글, 업데이트 글로 콘텐츠 확장이 가능한가

## 추천 3종

### 1. `Block Jam Blitz`

- 슬러그: `block-jam-blitz`
- 장르: 배치형 블록 퍼즐
- 포지셔닝:
  - `Tetris` 복각이 아니라, 모바일 친화적인 `배치 + 라인 클리어 + 콤보` 퍼즐로 잡는다.
  - 최근 퍼즐 트렌드의 `쉽게 시작해서 오래 붙는 반복 구조`에 가장 잘 맞는다.
- 추천 이유:
  - 드래그 기반이라 모바일 대응이 가장 쉽다.
  - 솔로 개발 기준으로 첫 MVP를 안정적으로 만들기 좋다.
  - 점수 경쟁, 일일 시드, 미션 확장이 자연스럽다.
- 우선순위: 1순위 MVP

### 2. `Snake Survivor`

- 슬러그: `snake-survivor`
- 장르: 생존형 아케이드 액션
- 포지셔닝:
  - 고전 `Snake`의 인지도를 살리되, `생존`, `웨이브`, `강화 선택`을 붙여 현대식 캐주얼 액션으로 재해석한다.
- 추천 이유:
  - 레트로 인지가 강해서 진입이 쉽다.
  - 단순 이동만으로도 숙련도 차이를 만들 수 있다.
  - 퍼크 선택과 적 패턴으로 재방문 동기를 만들기 좋다.
- 우선순위: 2순위

### 3. `Brick Shot Rush`

- 슬러그: `brick-shot-rush`
- 장르: 각도 발사형 브릭 슈터
- 포지셔닝:
  - 클래식 `Breakout`의 패들 조작 대신, `조준 후 발사` 중심의 현대식 브릭 브레이커로 잡는다.
  - 짧은 세션과 물리적인 만족감이 강해 모바일 웹에 더 잘 맞는다.
- 추천 이유:
  - 손맛이 직관적이고 시청각 만족감을 주기 쉽다.
  - 점수, 라운드, 콤보 구조가 명확하다.
  - 특별 블록, 추가 볼, 관통, 분열 같은 확장이 쉽다.
- 우선순위: 3순위

## 왜 이 3종인가

- 세 게임 모두 `레트로 감성`은 유지하지만, 조작은 모바일 친화적으로 현대화했다.
- 순수 복각형 `Breakout`보다 `Brick Shot Rush`가 현재 브라우저/캐주얼 유저 기대에 더 가깝다.
- `Block Jam Blitz`는 가장 낮은 구현 리스크로 첫 출시 품질을 올리기 좋다.
- `Snake Survivor`는 사이트의 대표성 있는 액션 게임 역할을 맡을 수 있다.
- 세 장르가 서로 달라 홈 화면과 `/game` 목록에서 선택지가 분명해진다.

## 보류하거나 피할 방향

- 순수 `Breakout` 복각
- `Tetris`와 너무 비슷한 명칭, 룰, 비주얼
- 키보드 전용 조작을 전제로 한 게임
- 긴 튜토리얼이 필요한 게임
- 계정 기반 성장이나 실시간 멀티플레이가 필수인 구조

## 추천 출시 순서

1. `Block Jam Blitz`
2. `Snake Survivor`
3. `Brick Shot Rush`

## 공통 출시 원칙

- 첫 공개는 게임 1종으로 시작한다.
- 게임 오버 후 즉시 점수 제출과 랭킹 확인이 가능해야 한다.
- 각 게임은 `1개 핵심 모드 + 1개 랭킹`만 먼저 붙인다.
- 일일 챌린지, 주간 시즌, 스킨, 추가 모드는 2차 확장으로 미룬다.

## 콘텐츠 확장 방향

- `Block Jam Blitz`
  - 배치 기본기
  - 콤보 유지법
  - 실수 줄이는 보드 운영 팁
- `Snake Survivor`
  - 초반 생존 팁
  - 추천 퍼크 조합
  - 적 패턴 공략
- `Brick Shot Rush`
  - 반사각 감각 익히기
  - 초반 추가 볼 우선순위
  - 위험 라인 관리법
